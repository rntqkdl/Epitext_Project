## 주요 파일 매핑

|    원본 (Epitext_Back)    |     통합 위치      | 기능         |
| :-----------------------: | :----------------: | ------------ |
| ai_modules/config/\*.json | src/models/config/ | 모델 설정    |
|  ai_modules/models/\*.py  | src/models/vision/ | 시각 모델    |
|   ai_modules/nlp/\*.py    |  src/models/nlp/   | NLP 모델     |
| ai_modules/\*\_engine.py  |    src/models/     | 엔진들       |
|       routes/\*.py        |  backend/routes/   | API 라우트   |
|      database/\*.py       | backend/database/  | DB 관리      |
|          app.py           |   backend/app.py   | FastAPI 메인 |

| 원본 (Epitext_Front) |        통합 위치         | 기능           |
| :------------------: | :----------------------: | -------------- |
|   src/components/    | frontend/src/components/ | React 컴포넌트 |
|      src/pages/      |   frontend/src/pages/    | 페이지         |
|       public/        |     frontend/public/     | 정적 파일      |
|     package.json     |  frontend/package.json   | npm 설정       |
