# Epitext 프로젝트 실행 가이드

## 🚀 Backend 실행

\\\ash
# 1. 가상환경 생성
python -m venv venv
venv\Scripts\activate

# 2. 패키지 설치
cd backend
pip install -r requirements.txt

# 3. 환경변수 설정
copy .env.example .env
# .env 파일 편집 (API 키 등)

# 4. 실행
python app.py
\\\

**접속**: http://localhost:8000

---

## 🌐 Frontend 실행

\\\ash
# 1. 패키지 설치
cd frontend
npm install

# 2. 환경변수 설정
copy .env.example .env
# .env 파일 편집 (Backend URL 등)

# 3. 실행
npm start
\\\

**접속**: http://localhost:3000

---

## 📦 전체 실행 (Docker)

\\\ash
docker-compose up -d
\\\

---

## ⚠️ 주의사항

1. **Python 버전**: 3.8 이상 필요
2. **Node.js 버전**: 16 이상 필요
3. **GPU**: CUDA 11.8 이상 권장 (선택)
4. **API 키**: Gemini API 키 필수
